História da Web

A 1ª Geração tem inicio com o Harvard Mark I em 1944, um computador eletromecânico que foi criado para realizar diversos tipos de cálculos complexos. Em 1946, meados da 2ª Guerra Mundial, foram criados os computadores Colossus para o lado da Alemanha, e Eniac para o lado dos Estados Unidos. Ainda nessa mesma época, houve o surgimento do termo bug, pois os computadores e suas válvulas esquentavam muito, atraindo insetos que pousavam nos equipamentos, e quebravam. A 2ª Geração de 1959 a 1965, com a evolução das válculas para os transistores que permitiu a redução no tamanho dos computadores. A 3ª Geração de 1965 a 1970, com o surgimento do circuito integrado composto pelo transistor, capacitor e resistor, possibilitou ainda mais a redução do tamanho dos computadores, bem como a eficácia desses equipamentos. Por fim, a 4ª Geração que teve início em 1971, é marcada pelo Personal Computer e o uso de mouses tornando a tecnologia cada vez mais popular e acessível. No entanto, há quem diga que já estamos vivenciando a 5ª Geração com o uso de inteligência artificial, criptomoedas e computadores quânticos.

A Internet surgiu com o propósito da comunicação, em 1958 o telégrafo permitiu a cominicação intercontinental. Em 1960, Licklider teorizou sobre uma rede onde todos poderiam se comunicar através dela. A Arpanet realizou estudos sobre nós e sistema de comunicação em pacotes, a primeira conexão ocorreu em 1969 entre a Universidade da Califórnia e o Instituto de Pesquisa de Stanford. Em 1989, Tim Berners-Lee, considerado o pai da Internet uniu o hipertexto com a rede TCP/IP e criou o primeiro navegador World Wide Web.

Entre os grandes nomes da tecnologia estão: Grace Hopper, a criadora da linguagem de programação de alto nível Flow-Matic, Joseph Licklider, iniciou os estudos da comunicação entre dois pontos distintos, Robert Kahn, um dos criadores do TCP/IP, Tim Berners-Lee, pai da internet, diretor da (W3C) que popularizou o HTTP e HTML, Marc Andreessen reinventou o navegador Nescape Navigator a partir do Mosaic, Bill Gates, fundador da Microsoft e Internet Explorer, tornando a internet mais acessível para todos, Steve Jobs, fundador da Apple, e responsável pela popularização do smartphone

O que são Clients?

Client é o dispositivo que está consumindo determinada informação na internet em formato HTTP, normalmente são representados por notebooks, desktops e celulares. São preocupações dos desenvolvedores, a responsividade e a compatibilidade entre navegadores.

Os navegadores são programas, utilizados para abrir ou executar arquivos como HTML, CSS e JS, que seguem padrões W3C.

Uma aplicação web utilizam a internet como meio de comunicação entre client x server sem a necessaidade de instalação do aplicativo.

O que são Servers?

Servers são computadores que fornecem a informação para os clients.

Os servidores podem estra dentro de uma rede local, ou estar disponível na internet, e serem servidores de de arquivos, segurança, streaming, email e web. O servidor proxy atua como intermediário de autenticação entre o cliente e a internet. O servidor firewall serve para bloquear o acesso externo, funcionando como uma barrreira de segurança. Os servidores web e email são servidores para sites e emails. o servidor database é um segundo servidor com o banco de dados acessado pelo primeiro servidor. O servidor intranet é o servidor web sem o acesso a internet, ficando disponível apenas para uma rede específica. O servidor DNS detém uma lista de nomes que redireciona para um IP.

Como é uma estrutura básica de um HTML?

Linguagem de Programação são instruções que geram softwares. O HTML não é uma linguagem de programação, é um arquivo de texto que possuem delimitadores que são interpretados pelo navegador.

Toda página HTML é composta por tags, algumas delas são:

  • head: coisas que vão carregar antes de exibir para o usuario
  • body: o que vai aparecer para o usuário
  • title: titulo da página
  • h1, h2, h3, etc.: títulos, subtítulos do texto
  • i: itálico
  • strong: negrito
  • input: campo de digitação
  • img: imagem
  • p: parágrafo
  • blockquote: citação
  • u: sublinhado
  • mark: marca texto
  • sup: texto escrito na parte superior da linha
  • sub: texto escrito na parte inferior da linha
  • hr: linha separadora, sem conteúdo
  • del: texto fica renderizado com uma linha entre o texto
  • abbr: define abreviação
  • ol: lista ordenada
  • ul: lista não ordenada
  • li: item da lista
  • a: âncora

Atributos são configurações aplicada em tags para que estas apareçam de forma diferente no navegador, algumas delas são:

  • type: tipo específico da tag input, como número, texto, etc.
  • id: identificador
  • style: comando para CSS
  • class: classe de padronização
  • src: atributo específico da tag img, com o endereço, link da imagem
  • width: atributo específico da tag img, largura da imagem
  • href: #identificador, link ou identificador de referência da âncora, atributo específico da tag a
  • target: atributo específico da tag a, _blanck abre o link em outra aba
  • title: atributo específico da tag a, atribui um título/texto que irá aparecer ao passar o mouse sobre a âncora
